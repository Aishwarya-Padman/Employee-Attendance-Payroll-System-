# Employee-Attendance-Payroll-System-
This project manages employee attendance and payroll using MySQL. It demonstrates database design, table creation, and writing advanced SQL queries for real-world HR analytics.The project covers:

✨ Project Highlights:

🗂️ Database Design: Relational tables for Employees, Attendance, and Payroll
🧹 Data Insertion & Cleaning: Structured data entry for accurate reporting
🔗 Complex Queries: Using joins, aggregations, subqueries, and window functions
💰 Payroll Analysis: Calculate total salary, overtime, departmental performance, and employee ranking
⚡ Query Optimization: Faster retrieval for large datasets
📊 Reporting: Generate performance and payroll reports for business decisions

🛠️ Tools Used: MySQL, MySQL Workbench

📌 Key SQL Concepts Practiced:

🔗 Joins: INNER, LEFT, RIGHT, FULL
📊 Aggregations: GROUP BY, HAVING
🎯 Window Functions: ROW_NUMBER, RANK, DENSE_RANK
🕵️ Subqueries & Correlated Subqueries
⚡ Query Optimization
